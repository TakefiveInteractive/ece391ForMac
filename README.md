# ECE391 Auto Setup For Mac


This is a repo created for ece391, since officially there is no setup guide provided for OS X. 

## Usage

```fish
git clone "https://github.com/Tedko/ece391ForMac.git"
```
then run 

```fish
cd ece391ForMac
./start
```
